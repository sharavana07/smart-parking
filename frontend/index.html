Project Overview

Our Smart Parking System aims to create a frictionless parking experience in layered parking spaces by using IoT technology to detect real-time parking slot availability. No more driving through multiple layers to find an empty spot!

Microcontroller

ESP32

Sensors

HC-SR04 Ultrasonic Sensors (2 units)

Data Transmission

WiFi (ESP32 to Web Server)

Interface

Web-based UI (Real-time parking slot status)

Programming

C++ (ESP32) & Web Technologies (UI)

How It Works

1

Detection

HC-SR04 ultrasonic sensors detect presence or absence of vehicles in parking slots

2

Processing

ESP32 processes sensor data and determines parking slot status

3

Transmission

Data is transmitted via WiFi to the web server

4

Display

Web interface updates in real-time to show current parking availability

Technical Details

ESP32 Specifications

  • Dual-core processor
  • Built-in WiFi and Bluetooth
  • Multiple GPIO pins for sensor connections
  • Low power consumption

HC-SR04 Sensor Details

  • Detection range: 2cm to 400cm
  • Resolution: 0.3cm
  • Measuring angle: 15 degrees
  • Operating voltage: 5V DC

Software Stack

  • ESP32: Arduino IDE with C++
  • Backend: Node.js
  • Database: MySQL
  • Frontend: HTML, CSS, JavaScript
