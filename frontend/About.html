About the Smart Parking System

The Smart Parking System is an IoT-based solution designed to modernize parking in multi-layered parking structures. It uses ultrasonic sensors to detect vehicle presence in parking slots and transmits real-time status to a web interface. This eliminates the need for manual verification of parking space availability, enhancing efficiency and convenience for users.

Our prototype currently features two active slots powered by ESP32 microcontrollers and HC-SR04 ultrasonic sensors. The system continuously updates the parking status and displays it on a user-friendly web interface built using HTML, CSS, and JavaScript.
